# rtt_pcb_defect_detect
    基于STM32H7的嵌入式机器视觉PCB缺陷检测系统 
    （代码见Release，完整zip包），pc_server.py为上位机WIFI-TCP通信链路代码
  摘要
  
  针对 PCB 生产行业人工目视检测效率低、漏检率高、传统 PC 机器视觉设
备成本高昂的痛点，本文设计一套基于 STM32H7 的嵌入式机器视觉 PCB 缺陷
检测系统，采用上下位机协同架构完成工业 PCB 自动化质检。上位机搭载奥普
特 GigE 工业相机采集高清 PCB 图像，完成图像格式转换与尺寸压缩，依托 
WiFi-TCP 通信协议向下位机传输标准化 RGB 图像数据流；下位机以 ART-Pi 
STM32H750 为主控，基于 RT-Thread 实时操作系统搭建多任务抢占式调度框架，
依托 TFLite Micro 推理引擎部署以 MobileNetV2 为骨干的 FOMO 轻量化目标
检测模型，完成 INT8 量化端侧 AI 推理，可精准识别 PCB 开路、短路、漏孔
三类典型缺陷并完成图像坐标映射。系统配套 TouchGFX 人机交互界面，实时展
示原图、缺陷标注框、检测置信度与统计信息，同时输出 GPIO 电平脉冲实现产
线工装硬件联动。本系统将图像预处理、深度学习推理、人机交互全部部署于嵌
入式单片机本地运行，无需工控机持续运行，兼顾低成本与实时性；RT-Thread 
通过优先级分层调度图像接收、AI 推理、界面刷新等任务，保障多流程并行稳
定运行。经实测，系统单张 PCB 图像检测耗时低于 300ms，缺陷识别准确率可
达 96% 以上，硬件成本远低于传统工业视觉设备，适用于小型 PCB 加工厂、实
验室研发样品检测等场景，为轻量化嵌入式工业机器视觉落地提供可行方案。

功能与特性 

本系统核心功能分为图像采集传输、嵌入式 AI 缺陷检测、人机交互、硬件
联动、Web网页仪表盘实时显示五大模块。上位机支持奥普特 GigE 工业相机实
时取图，通过自定义TCP协议向下位机传输RGB图像；下位机作为TCP客户端发
送拍照指令，接收图像后完成等比例缩放、查表式像素格式转换，适配 FOMO 模
型的输入规格。系统依托 RT-Thread 抢占式多任务调度机制，划分图像接收、
AI 推理、屏幕显示、外部中断四大独立任务，采用信号量与事件标志组完成任
务同步，规避多任务内存资源竞争问题。下位机部署由 Edge Impulse 训练生成
的量化 TFLite 模型，能够自主识别并定位 PCB 开路、短路、漏孔三类缺陷；
TouchGFX 交互界面支持手动自定义缺陷置信度判定阈值，同时可实时绘制缺陷
标注框、统计良品与不良品数量，并提供缺陷检测历史记录查询功能，方便追溯
检测样本;还可在Web端仪表盘查看缺陷数据。系统可响应工装 GPIO 外部中断
自动启动检测流程，AI 推理结束后输出电平信号联动流水线设备，形成全自动
质检闭环。拥有实时性强、硬件成本低、部署便捷的特点。

  主要技术
  
  1.上下位机协同视觉采集架构：采用奥普特GigE工业相机获取高清工业图
像，PC 端完成图像预处理压缩，减轻 STM32 内存与算力负担，自定义 TCP 通
信协议保障图像完整传输； 

2.RT-Thread 实时操作系统多任务调度：基于全抢占式优先级调度，分层处
理中断、图像接收、AI 推理、界面刷新任务，信号量同步实现任务有序执行，
兼顾实时响应与运算稳定； 

3.嵌入式轻量化深度学习方案：基于 MobileNetV2 骨干网络搭建 FOMO 目
标检测模型，经 INT8 量化压缩，通过 Edge Impulse 生成 TFLite 模型，依托 
TFLite Micro 在 STM32H7 本地完成端侧推理； 

4.轻量化图像预处理：采用查表法完成像素浮点转换，规避实时浮点运算占
用单片机算力，图像等比例缩放适配模型输入尺寸； 

5.可视化与人机联动：运用意法半导体TouchGFX平台，设计图形界面实时
展示缺陷检测结果，GPIO 硬件输出实现流水线传送带工装联动，完成自动化质
检闭环。
